# 🚀 Quick Start Guide - HP EX Intelligence Hub

## 5-Minute Setup

### 1. Open the Dashboard
- Open `index.html` in Chrome, Firefox, Safari, or Edge
- You should see the EX Intelligence Hub with a blue sidebar
- The AI Agent section loads by default

### 2. First Interaction - Chat with the Agent
**Copy one of these prompts into the chat box:**

```
"Sample 200 employees from Engineering who haven't been contacted in 6 months"
```

Press Enter or click Send. The agent will respond with:
- Number of eligible employees
- How many were excluded (recently contacted)
- Sample composition by department
- Coverage percentage

## 📊 What the Numbers Mean

### EX Score (0-10 scale)
- **8.0-10.0:** Excellent engagement (target zone)
- **7.0-8.0:** Good engagement (monitor)
- **6.0-7.0:** Concerning (intervention needed)
- **Below 6.0:** Critical (urgent action required)

### Response Rate (%)
- **75%+:** Excellent (employee interest high)
- **60-75%:** Good (healthy participation)
- **40-60%:** Concerning (low engagement signal)
- **Below 40%:** Problem (may indicate survey fatigue)

### Attrition Risk
- **HIGH:** 5-15% probability of leaving within 6 months
- **MEDIUM:** 15-40% probability (watch closely)
- **LOW:** <5% probability (engaged and stable)

### Sentiment Distribution
- **Positive:** Supportive comments (constructive feedback)
- **Neutral:** Factual observations (neither good nor bad)
- **Negative:** Critical feedback (areas needing improvement)

## 💬 Using the Chat Interface

### How the AI Agent Works

The agent responds to keywords in your prompts:

| **Keyword** | **Agent Action** |
|---|---|
| sample / select / identify | Sampling workflow |
| deploy / send / launch / survey | Deployment workflow |
| analyze / sentiment / feedback | Analysis workflow |
| predict / forecast / trend | Prediction workflow |
| historical / compare / evolved | Historical workflow |

### Example Conversation

```
YOU: "Sample 150 engineers who haven't been contacted in 6 months"

AGENT: Shows eligible count, excluded count, sample breakdown

YOU: "Great, now deploy our ESAT pulse to them next week"

AGENT: Confirms survey deployment with scheduling details

YOU: "When should we expect results?"

AGENT: "Based on typical response patterns, 7-10 days"
```

## 🛠️ Troubleshooting

### Charts not showing?
- Try refreshing the page (Ctrl+R)
- Make sure you're in the right section
- Give charts 1-2 seconds to load

### Data not updating?
- Click the relevant action button (Generate Sample, Load Data, etc.)
- Check that all required fields are filled in
- Look for the green success toast notification

### Prompts not responding?
- Make sure the prompt includes at least one relevant keyword
- Try using exact words from Quick Prompts
- Check the browser console for any errors
